# 🎮 GameGallery

**"Your personal game library, organized and visualized."**

---

## 📌 Overview

GameGallery is a modern full-stack web application that allows users to organize and showcase video games in a clean, alphabetical layout. Each game is displayed with cover art and can be tagged with statuses such as *Completed*, *Playing*, or *Favorite*.

This project is being built as a learning journey to explore modern web development using a real-world use case.

---

## ✨ Features

* 📚 Alphabetically sorted game library
* 🖼️ Visual display with game cover art
* 🏷️ Status tracking (Completed, Playing, etc.) *(planned)*
* ❤️ Favorite marking system *(planned)*
* ➕ Add new games dynamically *(planned)*

---

## 🛠️ Tech Stack

* Next.js (React framework)
* React
* Tailwind CSS
* Node.js

*(Database integration coming soon)*

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/gamegallery.git
cd gamegallery
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open http://localhost:3000 in your browser.
